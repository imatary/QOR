CodeQOR

CodeQOR is a shared library (.dll/.so) in modular builds and part of the unified shared library in monolithic builds
CodeQOR is platform (Compiler, OS, Architecture) independent
CodeQOR provides basic facilities to rest of the QOR libraries and to QOR applications.

You will need to link your QOR Application or QOR extension shared library with CodeQOR.

CodeQOR provides the framework Aspect Oriented Programming facilities within the QOR and QOR applications.
